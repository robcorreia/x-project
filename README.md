## Objetivo do Projeto

Projeto desenvolvido com o objetivo de renderizar uma lista de dados com grande escala de informações deixando o mais performático possível

## Techs usadas

O projeto foi desenvolvido com as tecnologias:

- React JS
- TypeScript
- Prettier
- ESlint
- Alma Design System
- Styled-Components
- React Hook Form
- Yup
